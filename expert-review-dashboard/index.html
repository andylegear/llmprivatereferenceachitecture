About This Evaluation

You are evaluating an FAQ chatbot that uses a small, CPU-only large language model (Gemma 2B) with retrieval-augmented generation (RAG) to answer questions about the Upstream trading platform.

For each item you will see:

  • Question — the user's question
  • Ground Truth — the correct answer from the knowledge base
  • Chatbot Response — what the system actually generated

Rate each response on three dimensions (5-point scale), flag any hallucinations, and optionally add comments. Every response includes a standard footer directing users to support@upstream.exchange — please ignore this boilerplate when scoring.

Questions are presented in random order. Choose how many to review below — 50 is recommended for statistical validity.

Your progress is saved automatically in your browser's local storage — you can close and return at any time. Use the same name each time to resume. You can stop early using the Finish & Export button in the header.

Evaluation design follows Van der Lee et al. (2021) best practices for human evaluation of generated text.
